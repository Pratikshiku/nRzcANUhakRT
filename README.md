### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、教练、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 健身课程
5. 健身器材
6. 会员卡信息
7. 新闻公告
8. 在线留言
9. 后台管理
10. 个人中心
11. 我的收藏 

**教练功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 课程预约管理
5. 训练计划管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 用户管理
4. 教练管理
5. 健身课程管理
6. 课程预约管理
7. 健身器材管理
8. 器材借用管理
9. 会员卡信息管理
10. 办卡信息管理
11. 训练计划管理
12. 教室信息管理
13. 在线留言
14. 系统管理（关于我们、系统简介、轮播图管理、新闻公告）
